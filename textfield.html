輸入框 Text-field

輸入框（Text-field）讓使用者可以在此輸入或編輯各種文字格式，包括數字。

原則

輸入框應該以可發現的，明確區分和高效率為原則。

  • 可發現的：輸入框應該是要突出並明確的指出使用者可以輸入的資訊；
  • 明確區分：輸入框應該要與其他元素清楚的區分出來；
  • 高效率：輸入框應該要讓使用者輕鬆理解必填資訊，也能夠提出任何錯誤的解決方式。

Spec

min-width：144px

信箱已使用，請試試其他信箱

$(function(){
  // 判斷有沒有值
  $("input").each(function(){
    $(this).on('change keyup copy paste cut', function(){
      if(!this.value) {
        $(this).parent().removeClass('hasValue');
      } else {
        $(this).parent().addClass('hasValue');
      }
    })
  })
  // 統計字數
  $('.form__group--countletter input').keyup(function() {
    $(this).siblings('i').children('span').html(this.value.length);
  })
});
