選取 Selection-controls

選取工具（Selection-controls）提供使用者選項來單個或多個選擇。

原則

選取工具應是熟悉的，快速瀏覽和高效率為原則。

  • 熟悉的：選取工具應該要已經出現在UI上有段時間，使用者能夠如預期般操作；
  • 快速瀏覽：當使用者快速瀏覽網頁而《天下》預選選取項目時，已選取項目的視覺需要比未選取項目更突出；
  • 高效率：選取工具要提供能讓使用者輕易比較出差異的選項。

<form action="#!" class="form">
  <label class="label--radio">
    <input type="radio" name="radio">
    <span class="label__radio__mark"></span>
    <span class="label__radio__txt"></span>
  </label>
</form>
.label--radio {
  display: block;
  position: relative;
  margin-bottom: 12px;
  font-size: 0;
  cursor: pointer;
  z-index: 1;
  @include user-select(none);
  span {
    display: inline-block;
    vertical-align: middle;
    @include transition(all 0.3s);
    &.label__radio__txt {
      padding-left: 1rem;
      font-size: 1.125rem;
    }
    &.label__radio__mark {
      position: relative;
      top: 0;
      left: 0;
      width: 20px;
      height: 20px;
      background-color: transparent;
      border: 2px solid $gray-500;
      @include border-radius(50%);
      &::before {
        @extend %before-after;
        position: absolute;
        top: 50%;
        left: 50%;
        background: $red-100;
        @include border-radius(50%);
        @include translate(-50%, -50%);
        @include opacity(0);
        @include transition(all 0.3s);
        z-index: -1;
      }
      &::after {
        content: "";
        position: absolute;
        display: none;
        top: 50%;
        left: 50%;
        width: 10px;
        height: 10px;
        background: $primary;
        @include border-radius(50%);
        @include translate(-50%, -50%);
      }
    }
  }
  input {
    position: absolute;
    width: 0;
    height: 0;
    overflow: hidden;
    opacity: 0;
    cursor: pointer;
    &:checked {
      ~ span.label__radio__mark {
        border-color: $primary;
        &::after {
          display: block;
        }
      }
    }

  }
  &:hover, &.active {
    input {
      ~ span.label__radio__mark {
        border-color: $primary;
      }
      &:checked {
        ~ span.label__radio__mark {
          &::before {
            width: 36px;
            height: 36px;
            @include opacity(1);
          }
        }
      }
    }
  }
}

Don't Do

Checkboxes 只能用在複選
