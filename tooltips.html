工具提示框 Tooltips

工具提示框（Tooltips）可以附加到任何元素。當使用者將鼠標懸停或點擊元素時，將顯示純文字提示使用者元素功能。

原則

工具提示框應以短暫出現，元素匹配和簡潔為原則。

  • 短暫的：鼠標懸停或點擊元素時，工具提示框將會短暫出現；
  • 匹配的：工具提示框會出現在元素的下方；
  • 簡潔的：工具提示框內只會有簡短的解釋文字。

How to Use

  • Default
<div class="tooltips" data-tooltips="說明文字">
	<!-- 放按鈕或icon -->
	</div>
$cubic: cubic-bezier(.64,.09,.08,1);
	.tooltips {
	position: relative;
	&:after {
		content: attr(data-tooltips);
		position: absolute;
		top: 100%;
		left: 50%;
		padding: 6px 8px;
		margin-top: 10px;
		max-width: 200px;
		color: $white;
		font-size: 14px;
		background: $gray-500;
		white-space: nowrap;
		visibility: hidden;
		@include opacity(0);
		@include transform(translateX(-50%) translateY(-2px));
		@include transition(opacity 0.2s $cubic, transform 0.2s $cubic);
		z-index: 2;
	}
	&-wrap {
		&:after {
		width: 200px;
		white-space: normal;
		}
	}
	&:hover, &.active {
		&:after {
		display: block;
		visibility: visible;
		@include opacity(1);
		@include transform(translateX(-50%) translateY(0));
		}
	}
	}
$(function(){
	// 當tooltips大於等於15字
	$(".tooltips").each(function(){
		if ($(this).data("tooltips").length >= 15 ) {
		$(this).addClass("tooltips-wrap");
		}
	});
	});
