天下數位全閱讀

1.天下數位全閱讀最新版本操作介紹

訂閱天下數位全閱讀服務，您可以透過天下雜誌 App 或天下雜誌官網使用天下數位全閱讀服務。

在您訂閱期間，可完整閱讀 App 與官網內的所有文章，不受閱讀篇數限制。為了享有完整閱讀權益，建議您在訂閱前先註冊、並登入天下雜誌會員。

天下雜誌 App

免費下載後，您可免費閱讀部分文章，並可使用四大功能幫助您探索感興趣的內容：

  1. 今日：《天下雜誌》編輯團隊每日策展重要議題，給沒時間又想快速掌握重點的您。
  2. 頻道：依天下雜誌文章內容分門別類，幫助您快速探索感興趣的內容。
  3. 雜誌：完整收錄《天下》紙本雜誌全文，並可回溯過去的經典好文。
  4. 收藏：一鍵收藏即可隨時接續再看，連貫零碎時間的行動閱讀。
天下雜誌官網
  1. 點選文章頁的圖示（ ），即可收藏您喜歡的文章。
  2. 點選任一作者，您可以追蹤您所喜歡的作者。
  3. 點選網頁右上角的人頭，進入會員中心，可看到您收藏的文章及作者。
  4. 點選任一文章頁的分享按鈕，付費訂戶可以分享您喜歡的文章給朋友完整閱讀，您的朋友閱讀該篇文章時將不會遇到付費牆。
2.怎麼開通天下數位全閱讀？

如果您透過網路訂購天下數位全閱讀，在您結帳之後，就可以馬上登入天下雜誌官網、天下雜誌 App 無限瀏覽文章。

如果您透過贈送取得、或從線下訂購天下數位全閱讀，在您付款之後隔天使用帳號及密碼，就可以登入天下雜誌官網、天下雜誌 App 瀏覽文章。

※ 若您沒有收到「天下數位全閱讀使用通知信」，可以先到垃圾郵件匣找找看，記得把《天下數位全閱讀》加入收件名單，以即時收到訂戶專屬的最新資訊喔！

3.海外地區可以使用天下數位全閱讀嗎？

在海外多數地區，只要能夠上網即可使用天下雜誌 web 與 App 服務。

提醒您，在中國地區因網路限制，有可能無法正常使用天下雜誌 web 與 App 服務，建議您先向所在地的「網路服務電信業者」反映此問題並瞭解原因後，再決定是否訂閱。

4.怎麼更改語言？

您可以在天下雜誌網站首頁左上方選擇「English」，就可以切換至天下雜誌英文網站；您也可以點選這裡進入天下英文網站，任意閱讀您感興趣的文章。

5.我是天下數位全閱讀訂戶，可以下載天下官網的文章嗎？

※ 因應天下雜誌數位營運策略之調整，天下知識庫點數已全面停止販售，感謝您的支持！您原持有的點數仍可正常使用，使用期限請見公告內容。

天下知識庫點數可以在天下雜誌群各個網站（含天下、康健）使用，只要找到您想要閱讀下載的文章，按下「下載」後輸入網路會員帳號密碼，就可以扣點下載 PDF 閱讀。

6.天下數位全閱讀有 iPad 版或平板 App 嗎？

我們正在開發 iPad／平板 App ，上線後會優先通知您。您可以先利用平板設備登入天下雜誌官網閱讀，就能夠無限暢讀您感興趣的文章。
