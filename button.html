按鈕 Button

按鈕（Button）讓使用者只需要點擊一下，行動呼籲（Call to Action，以下簡稱cta）及做出選擇。

原則

按鈕應該以能識別功能，易於找到和狀態清楚為原則。

  • 識別功能：要清楚地讓使用者知道，點擊按鈕時會發瘋什麼動作；
  • 易於找到：要在畫面之中，能讓使用者容易找到按鈕，當然也包括其他動作的按鈕；
  • 狀態清楚：一個完整的按鈕，應該要清楚地具備它所有的互動狀態。

Spec

min-width：144px

Outlined Button

Outlined buttons 是中強度的按鈕。在網頁中，雖然它仍是重要的cta，但重要程度並不是最高。

Default

Disabled

How to Use

  • Default
  • Disabled
<button class="btn btn--outlined" >搜尋網站的內容</button>
button, a {
  background-color: transparent;
  @include border-radius(0);
  @include appearance(none);
  &:focus {
    outline: 0;
  }
  &[disabled*='disabled'] {
    cursor: not-allowed;
  }
  &.btn {
    position: relative;
    display: inline-block;
    min-width: 120px;
    padding: 0.6875rem 1rem;
    font-size: 1rem;
    line-height: 1.25;
    text-align: center;
    word-break: keep-all;
    overflow: hidden;
    @include breakpoint(375px) {
      min-width: 135px;
    }
    @include breakpoint(768px) {
      min-width: 144px;
      padding: 0.8125rem 1rem;
    }
    + button.btn, + a.btn {
      margin-left: 0.5rem;
      @include breakpoint(768px) {
        margin-left: 1rem;
      }
    }
    &::before, &::after {
      @extend %before-after;
      position: absolute;
      @include opacity(0);
      z-index: -1;
    }
    &::before {
      left: 0;
      right: 0;
      top: 0;
      bottom: 0;
      @include transition(opacity 0.2s);
    }
    &::after {
      left: 50%;
      top: 50%;
      padding: 55%;
      width: 32px;
      height: 32px;
      @include border-radius(50%);
      @include transform(translate(-50%, -50%) scale(1));
      @include transition(opacity 2s, transform 0.5s);
    }
    &:hover {
      &::before {
        opacity: 0.04;
      }
      &:focus {
        &::before {
          opacity: 0.16;
        }
      }
    }
    &:focus {
      &::before {
        opacity: 0.12;
      }
    }
    &:active {
      &::after {
        opacity: 1;
        transform: translate(-50%, -50%) scale(0);
        transition: transform 0s;
      }
    }
    &:disabled {
      background-color: transparent;
      cursor: initial;
      &::before {
        opacity: 0;
      }
      &::after {
        opacity: 0;
      }
    }
    &--outlined {
      color: $surface;
      background-color: $white;
      border: 1px solid $surface;
      z-index: 1;
      &::-moz-focus-inner {
        border: none;
      }
      &:hover {
        color: $primary-variant;
        border-color: $primary-variant;
      }
      &::before {
        background-color: $red-100;
      }
      &::after {
        background-color: $red-100;
      }
    }
    &[disabled*='disabled'] {
      color: $white;
      background: $gray-200;
      border: 1px solid $gray-200;
      cursor: not-allowed;
    }
  }
}

Don't Do

不要截斷文字內容

文字內容不換行

請不要在按鈕加上陰影

請不要在按鈕中加icon

如果有空間並排，
不要將按鈕垂直擺放
