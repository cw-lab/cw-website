官網停止支援 IE 8 瀏覽器及 TLS 1.0 加密通訊協定

生效日期：2019年8月30日

親愛的讀者您好：

感謝您對天下雜誌的支持，為保障客戶權益與網路交易安全，本網站自2019年8月30日起，停止支援 TLS 1.0 加密通訊協定及 IE 8（含）以下瀏覽器，以提高網路傳輸資訊安全。

若您為IE瀏覽器之使用者，建議盡快更新至 IE 9 以上版本，或改使用 Chrome、Firefox 等最新版本之瀏覽器，並啟用 TLS 1.1 以上之加密通訊協定。

IE 9 以上啟用 TLS 1.1 教學：

  1. 開啟[IE] → 工具 (Alt + X) → 網際網路選項 → 進階 → 安全性
  2. 請務必確認開 TLS 1.0 / TLS 1.1 / TLS 1.2 選項
  3. 設定完成，請點選確定後，並重新啟動 IE 瀏覽器

造成不便之處，敬請見諒
